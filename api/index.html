class CollabFilt.Filt(missingaszero=False)¶

Item-based collaborative filter implementation in Python inspired by RecommenderLab R package. Uses sparse matrix storage format.

Example : 
>>> from CollabFilt import Filt
>>> filt = Filt(missingaszero=False)
>>> filt.addUser('u1', {'i1':5.0, 'i2':3.0, 'i3':2.5})
>>> filt.addUser('u2', {'i1':2.0, 'i2':2.5, 'i3':5.0})
>>> filt.addUser('u3', {'i1':2.5})
>>> filt.addUser('u4', {'i1':5.0, 'i3':3.0})
>>> filt.addUser('u5', {'i1':4.0, 'i2':3.0, 'i3':2.0})
>>> filt.dumpMatrix()
    id i1 i2 i3
    u1 5.0 3.0 2.5
    u2 2.0 2.5 5.0
    u3 2.5 None None
    u4 5.0 None 3.0
    u5 4.0 3.0 2.0
>>> filt.getUserCount()
    5
>>> filt.getItemCount()
    3
>>> filt.similarUsers({'i1':2, 'i3':1}, sim='euclid')
    [('u3', 0.6666666666666666),
     ('u5', 0.3090169943749474),
     ('u1', 0.22966848451216434),
     ('u4', 0.21712927295533244),
     ('u2', 0.2)]
>>> filt.predictRatings({'i1':2, 'i3':1}, sim='pearson')
    [('i2', 1.1666666666666667)]

pearson(user1, user2)¶

Calculate pearson similarity between two users. Because of the sparse storage format, we have to iterate through the key set union as opposed to the intersection. As required by Pearson, checks that there is more than a single factor overlap.

Note: Pearson coefficient is only defined if the variance of both vectors are nonzero.

Parameters:
  • user1 (dict) – user ratings
  • user2 – user ratings
Returns:
