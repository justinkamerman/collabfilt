Collabfilt

Collaborative filter implementation in Python inspired by RecommenderLab R package

Download this project as a .zip file Download this project as a tar.gz file

Description

Item-based collaborative filter implementation in Python inspired by RecommenderLab R package. Uses sparse matrix storage format. Currently only Euclidean and Pearson Coefficient distance metrics are implemented but adding new distance functions to the module should be relatively transparent to the rest of the module. A simple unit test regression accopanies the module.

Usage

The module consists of a single implementation file, CollabFilt.py, which is loaded into the Python interpreter at runtime with an import statement:


>>> from CollabFilt import Filt

To make sure your Python interpreter can locate the module, use one of the following methods:

  • Place a copy of collabfilt.py in your working directory.
  • Place a copy of collabfilt.py in your Python module path.
  • Add the location of collabfilt.py to the PYTHONPATH environment variable.
  • Append the location of collabfilt.py using the sys.path.append() function. 
    
        >>> import sys
        >>> sys.path.append("directory/containing/collabfilt.py")
        >>> from CollabFilt import Filt

Examples

An example within the repository uses CollabFilt to build a recommender application with the Book-Crossing dataset from the paper Improving Recommendation Lists Through Topic Diversification from the Institut f√ºr Informatik.

A simple example is shown below:


>>> from CollabFilt import Filt
>>> filt = Filt(missingaszero=False)
>>> filt.addUser('u1', {'i1':5.0, 'i2':3.0, 'i3':2.5})
>>> filt.addUser('u2', {'i1':2.0, 'i2':2.5, 'i3':5.0})
>>> filt.addUser('u3', {'i1':2.5})
>>> filt.addUser('u4', {'i1':5.0, 'i3':3.0})
>>> filt.addUser('u5', {'i1':4.0, 'i2':3.0, 'i3':2.0})
>>> filt.dumpMatrix()
    id i1 i2 i3
    u1 5.0 3.0 2.5
    u2 2.0 2.5 5.0
    u3 2.5 None None
    u4 5.0 None 3.0
    u5 4.0 3.0 2.0
>>> filt.getUserCount()
    5
>>> filt.getItemCount()
    3
>>> filt.similarUsers({'i1':2, 'i3':1}, sim='euclid')
    [('u3', 0.6666666666666666),
     ('u5', 0.3090169943749474),
     ('u1', 0.22966848451216434),
     ('u4', 0.21712927295533244),
     ('u2', 0.2)]
>>> filt.predictRatings({'i1':2, 'i3':1}, sim='pearson')
    [('i2', 1.1666666666666667)]
